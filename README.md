# sales-insights-Excel-dashboard
An interactive Excel Sales Dashboard built using Pivot Tables, Pivot Charts, Slicers, and Excel formulas to analyze sales performance, revenue, profit, customer segments, product categories, and regional trends.
## Dashboard Preview
[Dashboard Screenshot](your-screenshot-Excel file.png)
## Tools & Skills Used
- PivotTables & PivotCharts
- Slicers (interactive filtering)
- Interactive Charts
- Conditional Formatting
- Data Cleaning
- ## Key Insights
- West region generated the highest sales ($725K); South had the lowest ($391K)
- Technology was the most profitable category ($145K profit)
- Furniture had high sales but only a 2% profit margin — flagged as underperforming
- Overall: $2.3M total sales at a 12% profit margin
- ## How to Use
Download the .xlsx file and open in Excel — use the slicers to filter by 
region, quarter, or category and explore the dashboard interactively.
